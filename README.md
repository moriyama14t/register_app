# RegisterApp
 
node.js・Expressを用いた「ログイン認証システム」の実装（課題ポートフォリオ）

## 機能一覧
*  ユーザー登録機能
*  ログイン機能
*  ログアウト機能
*  ユーザー編集機能
*  ユーザー削除機能

## 技術一覧
*  プログラミング言語：Node.js v8.9.4
*  フレームワーク：Expess.js v4.16.1
*  データベース：MySQL
*  orm:sequelize
*  インフラ：heroku 
*  テンプレート：ejs

## 理解出来ていない・理解が曖昧な箇所
*  プロミス関数(わかりそうで理解できていない)
*  gitコマンド（push、commit等は利用できている）
 
## 工夫した点
*  ユーザー情報編集・削除追加
*  パスワードを忘れた際の秘密質問項目追加
